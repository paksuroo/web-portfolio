Project Description

Peas & Paws is a React-based pet adoption/appreciation platform that lets users browse animals, learn more about them, and connect with adoption services. It emphasizes user experience, intuitive navigation, and mobile responsiveness. I focused on clean layout, interactive elements, and consistent branding.

Key features include:
• Responsive design across devices
• Smooth navigation and transitions
• Clear information hierarchy (images, descriptions, calls to action)
• Performance optimizations and accessibility best practices

Technologies Used:
React CSS3 JavaScript Bootstrap
